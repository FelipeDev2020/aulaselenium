## 🎯 Sobre o Curso

O curso ensina a **criar testes automatizados do zero** para aplicações web usando **Selenium WebDriver em Java**.  
É totalmente prático e cobre desde os conceitos iniciais até tópicos avançados de escalabilidade:

- **Criar um projeto completo do zero**  
- **Construir um framework reutilizável** (com Page Objects, DSL, screenshots automáticos, etc.)  
- **Testar uma aplicação real** (sistema de gerenciamento financeiro)  
- **Executar testes em paralelo** e em **múltiplas máquinas** (Selenium Grid)  
- **Rodar na nuvem** (Saucelabs)

**Duração total:** 13h16min | **115 aulas** | **13 seções**

### O que estou aprendendo

- Criar testes automatizados com Selenium WebDriver  
- Interagir com elementos HTML (formulários, alerts, frames, pop-ups, tabelas dinâmicas)  
- Localizar elementos de forma robusta (XPath avançado)  
- Estratégias de sincronismo (esperas implícitas, explícitas e fluent wait)  
- Organizar o código com **Page Objects** e **DSL**  
- Criar um **framework completo** com funcionalidades extras  
- Gerenciar massa de dados nos testes  
- Executar testes **em paralelo** com JUnit  
- Configurar e usar **Selenium Grid** (múltiplos navegadores e SOs)  
- Executar testes na nuvem com **Saucelabs**
